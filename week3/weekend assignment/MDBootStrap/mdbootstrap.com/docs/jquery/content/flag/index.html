Flags

Bootstrap flags

A flag is used to represent a political state

To start working with Flags, please see "Getting Started" tab on this page.

Note: Flags integration is available from version 4.8.1 (released 13.05.2019).

Definition

Flag

A flag can use the two digit country code, the full name, or a common alias

  • HTML


        <i class="ae flag"></i>
        <i class="france flag"></i>
        <i class="myanmar flag"></i>

Types

Many flag sets use SVG to render flags at multiple sizes. However svg flags usually appear blurry or render improperly at smaller sizes. MDB default theme uses an image sprite solution which provides crisper icons, but without the ability to resize

Bootstrap flags - getting started : download & setup

Download

All the components and features are part of MDBootstrap package.

MDBootstrap (Material Design for Bootstrap) is a free (MIT Licensed) framework combining Material Design and the newest Bootstrap 4.

Click the button below to go to Download Page, where you can download MDBootstrap package.

MDBootstrap Download MDBootstrap About

Setup

After downloading and extracting MDB package, open index.html file.

Then, below the linked CSS files add flags file.


<!-- MDBootstrap flags -->
<link href="css/addons/flag.min.css" rel="stylesheet">

If you need additional help, have a look at the video tutorial below.

The video below refers to the other plugin, but it does not matter. The installation process is the same for all plugins.

Compilation

To reduce a weight of MDBootstrap package, you can compile your own, custom package containing only components and features you need.
