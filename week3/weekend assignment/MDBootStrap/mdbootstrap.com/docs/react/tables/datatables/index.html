React Bootstrap Datatables

React Datatables - Bootstrap 4 & Material Design. Examples & tutorial.

Tables let you aggregate a huge amount of data and present it in the clear and orderly way. React Bootstrap Datatables provide additional benefits like responsiveness and the possibility to manipulate the styles of the tables. With Datatables you can use an advanced options like sorting, searching or pagination. Also, our Datatables are scallable and responsive.

            sort: String (Optional, 'asc'/'desc'/'disabled', describes the initial sorting directory, or disables column sorting),
            attributes: {
              "attribute": "value"
            } : Object collection (Optional, sets html attributes of the element | f.e. "aria-controls", "aria-label")

          }
        ],
        rows: [
          {
            field: String (corresponds column's field, contains cell data),
            field: String,
            field: String,
            ...
            clickEvent: yourClickHandler
          }
        ]
      }
